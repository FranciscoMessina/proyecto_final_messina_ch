# proyecto_final_messina_ch

Proyecto final para el curso de Desarrollo de videojuegos de Coder House

# El juego:

- Juego por niveles con el objetivo principal de derrotar a los enemigos para pasar de nivel.
- El jugador puede moverse con WASD(Rotar la camara siguiendo al mouse) y tiene dos tipos de ataque, uno de area y otro de proyectil.
- Hay 3 tipos de enemigos 2 que te persiguen y atacan, uno mas resistente que el otro. Y uno que te dispara a la distancia.

- Se van a agregar obstaculos destruibles en el mapa.

- Si logramos cumplir estos objetivos y sobra tiempo, pensamos en la posibilidad de agregar variaciones en los niveles, como un puzzle, pickups o algo similar.

# KNOWN BUGS:

- Nos falta solucionar algunos temas de la camara, no funciona perfecto.

- Al golpear a un enemigo se destruye automaticamente, se le va a agregar animaciones y vida para que no mueran de un solo ataque, dependiendo del tipo de enemigo.

- El movimiento de los enemigos tiene que ser optimizado, ya que en este momento no funciona fluidamente.
